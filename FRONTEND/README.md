# Tienda Belleza - Frontend Qt (C++)

Proyecto: interfaz gráfica con Qt (Widgets) para gestionar productos de belleza.

## Estructura
- `src/` : código fuente
- `include/` : headers
- `ui/` : archivo .ui (Qt Designer)
- `data/productos.txt` : "base de datos" en texto

## Requisitos
- CMake >= 3.16
- Qt6 (o Qt5, adaptar CMake)
